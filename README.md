# Curezy - Healthcare Connector Platform

**Curezy** is a comprehensive healthcare management platform that connects patients, doctors, vendors, and delivery partners in a unified ecosystem. It enables seamless appointment scheduling, health tracking, medicine delivery, and medical record management.

## ‚ú® Features

- **Patient Portal**: Health checkups, appointment booking, medicine ordering, medical records, and emergency services
- **Doctor Dashboard**: Patient management, appointment handling, and medical record access
- **Vendor Management**: Medicine inventory management, order tracking, and catalog management
- **Delivery Tracking**: Real-time delivery status, route optimization, and earnings tracking
- **Admin Panel**: User management, medicine catalog administration, and system analytics
- **Health Records**: Comprehensive medical history and health check tracking
- **Real-time Notifications**: Instant updates on appointments, orders, and deliveries
- **Map Integration**: Location-based services with Leaflet and Mapbox
- **Responsive Design**: Fully responsive UI for all devices
